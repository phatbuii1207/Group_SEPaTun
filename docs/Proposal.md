# PROPOSAL DỰ ÁN
## QUẢN LÝ CỬA HÀNG THÚ CƯNG

### 1. Giới thiệu
Dự án xây dựng chương trình quản lý cửa hàng thú cưng sử dụng ngôn ngữ Java
trên môi trường Console, phục vụ cho môn học PRO192.

### 2. Mục tiêu
- Quản lý thông tin thú cưng trong cửa hàng
- Quản lý sản phẩm liên quan đến thú cưng
- Hỗ trợ lưu và đọc dữ liệu từ file

### 3. Vấn đề cần giải quyết
Việc quản lý thủ công gây khó khăn và mất thời gian.

### 4. Giải pháp đề xuất
Xây dựng chương trình Java Console có menu cho phép quản lý thú cưng và sản phẩm.

### 5. Đối tượng sử dụng
Chủ cửa hàng thú cưng quy mô nhỏ.

### 6. Phạm vi dự án
Chạy trên Java Console, không có giao diện đồ họa.
